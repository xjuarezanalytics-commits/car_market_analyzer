# 🚗 Car Market Analyzer

An interactive web dashboard to explore 51,000+ U.S. vehicle listings. Filter by brand, condition, fuel type, price range, and model year to uncover pricing patterns and market trends.

## 🚀 Live Demo
[View on Render](#) ← replace with your Render URL

## ✨ Features
- **Sidebar filters** — brand, condition, fuel, price range, model year
- **Live KPI cards** — vehicles found, median price, avg odometer, avg days listed
- **6 interactive charts** — price & odometer distributions, scatter by condition, box plots, top brands by price, fuel type breakdown, price trend by year
- **Raw data toggle** — inspect a sample of filtered records

## 🛠 Tech Stack
Python · Pandas · Streamlit · Plotly · EDA

## ⚙️ Run Locally
```bash
git clone https://github.com/YOUR_USERNAME/car-market-analyzer
cd car-market-analyzer
pip install -r requirements.txt
streamlit run app.py
```
